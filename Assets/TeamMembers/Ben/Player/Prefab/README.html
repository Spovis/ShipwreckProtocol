Description: This player controller is built for creating a scalable, easy to use 2D platformer. It has several built-in states and transitions to ensure the correct behavior when playing the game. By using a state machine, new states can be easily added with unique functionality without affecting existing states.
The Player asset includes the following:
  • Player Input Management with controller and mobile support
  • Scalable State Machine
  • Drag-and-drop player controller
Features:
  • On-screen-controls that can be enabled or disabled easily
  • Easily transition between predefined states that gives the player its behaviors
  • Easily add new states to the state machine with their own logic to suit your game
Requirements: Unity 2022.3.43f1 or later
