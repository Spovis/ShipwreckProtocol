Description: This asset allows you to add a premade settings menu to your game. This menu includes audio sliders and an invincibility checkbox.

This feature will allow you to easily integrate a settings menu into your game without the hassle of writing code.

The Settings Menu asset includes the following:

  • Settings Menu Prefab
  • SliderBase Class Script
  • Music Volume Slider Class Script
  • SFX Volume Slider Class Script
  • Settings Manager Class Script

The Settings Menu asset requires(not included) the following:

  • An Audio Manager with both Music and SFX
  • 2 volume sliders
  • Invincibility checkbox
  • Back Button

Features:

  • Settings Menu Title
  • Invincibility checkbox
  • 2 Volume Sliders
  • Back Button

Requirements: Unity 2022.3.42f1 or later
